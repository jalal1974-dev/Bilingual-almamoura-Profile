Technical Notes

All text must be stored in a translations object with en and ar keys, toggled by a lang state variable
When lang === 'ar': set document.documentElement.dir = 'rtl', use Arabic font, right-align text
When lang === 'en': set document.documentElement.dir = 'ltr', use English font, left-align
Use useEffect to update dir and lang attribute on <html> on toggle
Animate section entries with IntersectionObserver or a CSS class toggled on scroll
No external UI libraries — pure React + inline styles or a single <style> block
Gold accent #C09030 for all highlights, borders, icons, and CTAs
The language button should be always visible (fixed or sticky in nav)
